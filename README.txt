Navigoals is a digital agenda designed to help you efficiently plan, track, and evaluate your tasks and goals. It provides a structured system to stay on top of daily, weekly, and monthly objectives while boosting productivity.

---

## Features

- Task Categorization: Organize tasks into categories like Work, Study, Social, and Personal.
- Daily, Weekly, and Monthly Views: Gain clear insights into your short- and long-term objectives.
- Efficiency Metrics: Automatically calculate productivity to track your progress.
- Watch Lists: Monitor high-priority or long-term tasks and goals.
- Flexibility: Handle recurring tasks, unscheduled tasks, and task duplication with ease.
- Open Source: Fully customizable and free to use.

---

## What Problem Does Navigoals Solve?

Navigoals helps users overcome common challenges such as:

- Forgetting tasks or deadlines.
- Failing to measure progress over time.
- Struggling to adapt plans as priorities change.

It provides a structured, user-friendly solution for managing tasks and goals in a busy world.

## Getting Started

### Prerequisites

You will need Python 3.x installed on your system.
The `datetime` module for handling dates (included in Python's standard library)
The `tabulate` library for displaying tables. Install it by running:
    pip install tabulate


### Installation and Usage

1. Download the project files.
2. Ensure you are in the project directory.
3. Initialize the database by running:
    python initialize_db.py
4. Launch the application by running:
    python main.py
